# Vendor Client Management System (VCMS)

A modern role-based Vendor Client Management System built using the MERN stack.

## 🚀 Features

- Role-Based Authentication
  - Super Admin
  - Admin
  - Vendor

- Dashboard
  - Statistics Cards
  - Revenue Chart
  - Client Growth Chart
  - Recent Vendors
  - Recent Clients
  - Pending Tasks
  - Notifications
  - Quick Actions

- Vendor Management
- Client Management
- Employee Management
- Project Management
- Attendance Management
- Leave Management
- Payroll Management
- Task Management
- Reports
- Notifications
- Settings

# Installation

## Clone Repository

```bash
git clone https://github.com/selvavigneshm19/Vendor-Client-Management-System.git
```

```bash
cd Vendor-Client-Management-System
```

---

# Install Frontend

```bash
cd client
```

```bash
npm install
```

Run Frontend

```bash
npm run dev
```

Frontend runs on

```
http://localhost:5173
```

---

# Install Backend

Open another terminal

```bash
cd server
```

```bash
npm install
```

Create

```
.env
```

Example

```env
PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret_key
```

Run Backend

```bash
npm run dev
```

Backend runs on

```
http://localhost:5000
```
